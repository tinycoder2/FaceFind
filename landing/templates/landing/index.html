Frequently Asked Questions

  • What happens after I register a Missing Person?

    Once you have successfully registered a missing person, we will contact you for a background check followed by which, we generate a unique Face ID for the missing person using Azure's Face API. When volunteers report a suspected missing person, we verify and generate a Face ID the same way. We then use Azure's Find Similar API to identify a potential match with our database of missing person Face IDs. If a match is found we will report to the local authorities and contact you.

  • What happens after I report a Suspected Missing Person?

    Once you have successfully reported a suspected missing person, we will cross-verify the report to check its validity. We then generate a unique Face ID for the missing person using Azure's Face API, followed by which, we use Azure's Find Similar API to identify a potential match with our database of missing person Face IDs. If a match is found we will report to the local authorities and contact the family/guardian.

  • How does Azure's Face API work?

    Firstly, it detects a face in the given image and generates a unique identifier string for the detected face in the image. Then to check if two images contain the same person, we use the Find Similar API, that searchs for similar-looking faces given a target Face ID and Face ID array. The API returns a JSON object containing the matched Face ID from the given array along with the confidence score.
